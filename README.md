# GST Calculator

A modern and responsive GST Calculator built using HTML, CSS, and JavaScript.

## Features

 Calculate GST instantly

 Supports 5%, 12%, 18%, and 28% GST rates

 Quick GST selection buttons

 Responsive design for mobile and desktop

 Dark Mode support

 Modern UI with hover animations

 Real-time GST calculation results

## How to Use

1. Enter the amount.
2. Select a GST rate.
3. Click "Calculate GST".
4. View GST amount and total amount.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
